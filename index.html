Web-ToolBox-OverLay

Suite de accesibilidad web gratuita, personalizable y segura. Modifica textos, contrastes, colores y cursores en cualquier página, respetando tu privacidad.

🛡️ Aislamiento Total

Nuestra interfaz usa Shadow DOM. Ningún diseño de las webs que visitas romperá el menú.

💾 Persistencia

Guarda el tamaño y posición de tu botón flotante de forma automática y local.

🚀 Cero Dependencias

Ligero, sin llamadas a APIs externas. Funciona 100% offline y Respeta tu privacidad.

📖 Adaptable

Fuente OpenDyslexic, modo alto contraste, y cursores tamaño XL para todos.

Instalar Userscript Ver código fuente en GitHub
⚠️ Requisito Importante: Debes tener instalada la extensión Tampermonkey (o similar como Violentmonkey) en tu navegador antes de instalar el script.
